# Grocery-Billing-System-Rahul-202501100700113-B
Problem Statement
A grocery store wants to calculate the total cost of items purchased by a customer.

The system should accept price input for 5 different items.
Each item can be purchased in multiple quantities.
Calculate the total cost of all items.
Apply a 10% discount if the total amount exceeds Rs. 100.

Display:
Original Total
Discount (if applicable)
Final Payable Amount

Objectives
Practice Python input/output operations
Use loops for repetitive tasks
Apply conditional statements
Perform arithmetic calculations

Approach
Initialize a variable total to store the total bill amount.
Use a for loop to input price and quantity for 5 items.
Multiply price Ã— quantity to calculate each item total.
Add each item total to the overall total.
Check if total > 100:
If yes, apply 10% discount.
Otherwise, no discount.
Display original total, discount, and final payable amount.
